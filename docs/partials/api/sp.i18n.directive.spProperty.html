spProperty
directive in module sp.i18n

Description

Directive for displaying properties from the sp.i18n.spProperties service. Note: this directive also support binding with the curly braches notation ( {{ }} )

Usage

as attribute
<ANY sp-property
     key="{String}"
     [identifier="{String}"]
     [locale="{String}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
keyString

The property key to display

identifier
(optional)
String

The properties identifier

locale
(optional)
String

The locale
